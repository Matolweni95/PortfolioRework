Project Purpose and Goal

The goal is to create a fully functional full-stack application to solidify my learning.

Problems and Thought Process

I have attempted a full-stack application like this before. At the time, I can admit to being in tutorial purgatory and purely coding along. I barely had any problems I couldn't tackle without going back to the tutorial. With this project, I’m going off of my own knowledge, documentation and stack overflow for when I run into any problem.

The first phase, the User Interface is complete ( I will do further refinements to the styling and layout later ). The next phase of the project is implementing a backend, As previously stated the application will be a MEAN stack and I need to get familiar with Mongo and Express. I have already started creating clusters and learning how to use my CLI to start up Mongo. The next step is understanding how to use relationships from there, create my schema then use Node to communicate with Mongo.

It’s still a rough thought process and can change as I further the tools I want to use.
