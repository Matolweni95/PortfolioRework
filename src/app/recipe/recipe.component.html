Project Purpose and Goal

The goal was to solidify my understanding of margins and get familiar and comfortable using flexbox.

Problems and Thought Process

One problem I had with this project was having to use a lot of manual properties to move things around. It feels like trial and error. I believe it's not best practice and that there are more efficient ways to get elements positioned accordingly. To achieve the side-by-side look, I used Flex, understanding that the elements will be next to each other. The next step would be to resize and then add a z-index property to place one element over the other. And then add margins to move it into position.
