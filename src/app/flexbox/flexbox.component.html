Project Purpose and Goal

There was no real purpose for this project. I was already comfortable using flex and had been using it as my main method to center my elements. The idea was inspired by memes about backend developers finding CSS a challenge, so I built a bit of a refresher.

Problems and Thought Process

The project was simple, I needed to make a form to get data and then manipulate it as needed. Then made a container and had 2 elements (red and blue) within it. I used Flexbox to center both form and container. I opted to use a function to get to data as I felt it'd be easier for what I needed. For the purpose of the project, I restricted input ability, the required text is selectable from drop-down boxes. Using ngModel directive, I was able to get the selected inputs. From there I made a function that is called by the Test Button on the form. Within the function, I used DOM styling to add the properties to the container to showcase the effects of Flexbox.

Moving forward I would like to add more flex properties, at the moment you can explore flex-direction. I want to also show how to use justify-content to add space around your elements
